#Mario

##iOS Game: Mario

Created own mario game after learning Zombie Conga Game

####Language:
* Objective-C
* Swift 3/4
* xCode

####Frameworks:
* UIKit
* Foundation
* CoreGraphics 

####General Rules:
* Eating an apple gains you one point
* Eating a strawberry gains you two points 
* Touching pipes leads you to game over
